# Awesome Project Build with TypeORM

Steps to run this project:

1. Run `npm i` command
2. Run `docker-compose up` command
3. Run `npm start` command


## 以下のコマンドで一括作成
「環境作成コマンド」  
npx typeorm init --name typeorm-study --express --docker --database postgres
